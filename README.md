# Todo App with Chatbot

A Python-based todo application featuring an interactive chatbot that helps manage your daily tasks.

## Features

- Add new todo tasks
- View all tasks with completion status
- Mark tasks as completed
- Delete tasks
- Save tasks in a JSON file
- Interactive chatbot that guides users through the application
- Natural language processing for common commands

## How to Run

1. Make sure you have Python 3 installed on your system
2. Download or clone this repository
3. Navigate to the project directory
4. Run the application:

```bash
python main.py
```

## Available Commands

- `add task` or `add`: Add a new task
- `show tasks` or `list`: View all tasks
- `complete task` or `done`: Mark a task as completed
- `delete task` or `remove task`: Delete a task
- `exit` or `quit`: Exit the application

## File Structure

- `main.py`: Main application logic
- `chatbot.py`: Chatbot functionality
- `tasks.json`: Local storage for tasks
- `.gitignore`: Files to be ignored by Git

## Technologies Used

- Python 3
- Standard library only (no external dependencies)
